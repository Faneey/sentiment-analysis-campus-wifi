# sentiment-analysis-campus-wifi

# Student Sentiment Analysis on UPI YPTK Padang WiFi Services Using Machine Learning

## Overview

This project analyzes student opinions regarding WiFi services at UPI YPTK Padang using machine learning techniques. The system classifies feedback into Positive, Neutral, and Negative sentiment categories and evaluates multiple classification models to identify the best-performing approach.

## Technologies

- Python
- JupyterLab
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn
- TF-IDF Vectorizer

## Dataset

Student feedback and reviews regarding WiFi services at UPI YPTK Padang collected through surveys and questionnaires.

## Key Results

- Classified student feedback into Positive, Neutral, and Negative sentiments.
- Compared multiple machine learning algorithms including Naive Bayes, SVM, KNN, Decision Tree, and Random Forest.
- Evaluated model performance using accuracy scores and confusion matrices.
- Generated insights regarding student satisfaction with campus WiFi services.
